This is a Vite + React project.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the app by modifying files under `src/`. The page auto-updates as you edit the file.

## Environment

Create a local `.env` file with these values:

- `VITE_API_URL=https://project-management-system-wacw.onrender.com/api/v1`
- `VITE_SOCKET_URL=https://project-management-system-wacw.onrender.com`
- `VITE_API_DEBUG=true`

The same values should be added in Vercel project settings for Production and Preview.

## Learn More

To learn more about Vite and React, take a look at these resources:

- [Vite Documentation](https://vite.dev/guide/)
- [React Router Documentation](https://reactrouter.com/)

## Deployment

Deploy the frontend on Vercel and make sure these environment variables are set:

- `VITE_API_URL`
- `VITE_SOCKET_URL`
- `VITE_API_DEBUG`

The backend API is hosted at [https://project-management-system-wacw.onrender.com](https://project-management-system-wacw.onrender.com) and the API base is [https://project-management-system-wacw.onrender.com/api/v1](https://project-management-system-wacw.onrender.com/api/v1).
